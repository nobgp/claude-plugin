# nobgp Claude Code Marketplace

A Claude Code plugin marketplace for managing nobgp network infrastructure.

## Adding This Marketplace

```bash
claude plugin marketplace add nobgp/claude-plugin
```

## Available Plugins

### nobgp

Manage nobgp networks, nodes, and services directly from Claude Code.

```bash
claude plugin install nobgp@nobgp-marketplace
```

See [plugins/nobgp/README.md](plugins/nobgp/README.md) for details.
